| 용어 | 뜻 |
| --- | --- |
| 파트너센터 | 병원이 병원 정보, 이벤트, 상담, 예약, 광고, 후기 답글을 관리하는 웹 화면 |
| 이벤트 | 병원이 등록하는 프로모션성 진료·수술 상품 |
| 검수 | 자신있나가 의료광고, 가격, 이미지, 조건을 확인하는 절차 |
| 광고잔고 | 광고 집행에 사용할 수 있는 병원 잔액 |
| 유료 충전 포인트 | 병원이 실제 결제 또는 세금계산서 발행으로 충전한 광고비 |
| 무료 지급 포인트 | 프로모션, 입점 혜택, 보상 목적으로 지급된 무상 포인트 |
| 일예산 | 하루 동안 사용할 수 있는 광고비 한도 |
| 상담수 | 전화, 채팅, 앱예약, 콜백요청 등 시스템에 기록되는 상담 행동 |
