자신있나 광고는 병원이 원하는 노출 목적에 따라 상품을 선택해 운영할 수 있어요.

## 광고 상품

| 상품 | 설명 | 과금 방식 |
| --- | --- | --- |
| 일반 CPV | 일반 유입으로 이벤트·병원·의사 상세를 조회한 경우 | 상세조회당 과금 |
| CPC 광고 | 광고 클릭 후 상세로 진입한 경우 | 유효 클릭 기준 과금 |
| CPV 프리미엄 입찰 | 프리미엄 영역에서 이벤트 상세로 진입한 경우 | 프리미엄 상세조회당 과금 |
| 보장형 광고 | 배너, 팝업, 푸시, 카테고리 스폰서, 후기 콘텐츠 광고 | 정해진 금액 과금 |
| 프리미엄 병원 | 병원 노출 강화를 위한 월정액 상품 | 월 정기 과금 |

## 추가 과금되지 않는 행동

상담신청, 전화 버튼 클릭, 채팅상담, 앱예약, 콜백요청 자체에는 추가 광고비가 붙지 않아요.

## 중복 과금 기준

하나의 사용자 행동에는 하나의 광고비만 과금돼요. CPC 광고로 상세에 들어온 경우 일반 CPV가 추가 과금되지 않아요. 보장형 광고나 프리미엄 병원 카드 유입에도 별도 CPV가 붙지 않아요.
