광고 리포트는 사용금액만 보는 화면이 아니에요. 어떤 광고가 상담과 예약으로 이어졌는지 확인하는 기준이에요.

## 리포트에서 먼저 볼 항목

| 항목 | 확인 방법 |
| --- | --- |
| 노출·조회 흐름 | 광고가 실제로 보여지고 상세 화면으로 이어지는지 확인해요. |
| 상담 연결 | 전화, 채팅, 예약, 콜백 요청이 같은 기간 안에서 늘었는지 확인해요. |
| 예약 연결 | 상담 이후 예약 요청 또는 확정으로 이어지는지 확인해요. |
| 사용금액 | 비용만 보지 말고 상담과 예약 흐름을 함께 비교해요. |
| 지역별 흐름 | 특정 지역에서 상담 품질이 낮으면 지역 범위와 이벤트 조건을 다시 봐요. |

## 상담수 기준

상담수는 실제 통화 완료만 의미하지 않아요. 전화 버튼 클릭, 전화상담 신청, 채팅상담, 앱예약, 콜백요청처럼 시스템에 기록되는 상담 행동을 기준으로 볼 수 있어요.

## 성과 개선 방법

- 조회는 높은데 상담이 낮으면 이벤트 제목, 가격, 이미지, 추가 비용 안내를 확인해요.
- 상담은 높은데 예약이 낮으면 콜백 속도와 상담 스크립트를 확인해요.
- 특정 지역 성과가 낮으면 광고 지역을 조정해요.
