광고는 신청만 하면 바로 노출되는 것이 아니에요. 광고 소재, 랜딩 이벤트, 광고잔고, 지역, 예산을 확인한 뒤 승인돼요.

## 신청 전 체크리스트

- 광고에 연결할 이벤트가 승인 상태인가요?
- 이벤트 가격과 이미지가 최신인가요?
- 광고 소재에 의료광고 금지 표현이 없나요?
- 광고 지역이 병원 진료권역과 맞나요?
- 광고잔고가 충분한가요?
- 일예산이 너무 낮게 설정되어 있지 않나요?

## 광고가 노출되지 않는 주요 이유

| 사유 | 확인할 것 |
| --- | --- |
| 승인 대기 | 광고 검수 결과를 기다려 주세요 |
| 반려 | 반려 사유에 맞게 문구, 이미지, 가격을 수정해 주세요 |
| 잔고 부족 | 광고잔고와 포인트를 확인해 주세요 |
| 일예산 소진 | 다음날 초기화 또는 예산 조정이 필요해요 |
| 랜딩 이벤트 변경 | 이벤트가 재검수 상태인지 확인해 주세요 |
| 지역 불일치 | 광고 지역과 환자 탐색지역 조건을 확인해 주세요 |
