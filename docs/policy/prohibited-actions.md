아래 행동은 병원 신뢰와 환자 보호에 영향을 줄 수 있어요. 반복되면 콘텐츠 수정 요청, 노출 제한, 광고 중지, 계정 확인 등의 운영 조치가 진행될 수 있어요.

## 금지 행동

| 구분 | 하면 안 되는 행동 |
| --- | --- |
| 허위 정보 | 실제와 다른 가격, 진료시간, 의사 정보, 이벤트 조건을 등록해요 |
| 과장 광고 | 효과, 통증, 회복, 만족을 보장하는 표현을 사용해요 |
| 개인정보 | 환자 연락처, 상담 내용, 신체사진을 목적 외로 사용해요 |
| 어뷰징 | 직원·지인을 통한 허위 후기나 비정상 클릭을 유도해요 |
| 우회 운영 | 검수 반려 문구를 이미지나 다른 문서에 다시 넣어요 |
| 외부 전송 | 상담 이미지를 개인 메신저나 외부 저장소로 옮겨요 |

## 운영 제한이 걸릴 수 있어요

1. 자신있나가 수정 요청을 안내해요.
2. 병원이 문구, 이미지, 가격, 조건을 수정해요.
3. 수정 전까지 해당 콘텐츠나 광고 노출이 제한될 수 있어요.
4. 반복되면 추가 검토 후 이용 범위가 제한될 수 있어요.

## 병원 내부에서 예방해요

- 콘텐츠 등록 전 담당자 1명이 가격과 문구를 최종 확인해요.
- 이벤트 이미지와 파트너센터 입력값이 같은지 확인해요.
- 상담 담당자가 앱에 보이는 조건을 그대로 안내하도록 공유해요.
- 환자정보 접근 권한은 필요한 직원에게만 부여해요.
