운영 중 자주 생기는 문제는 대부분 상태, 잔고, 검수, 담당자 기준을 확인하면 해결할 수 있어요.

## 광고가 보이지 않아요

1. 광고잔고가 부족하지 않은지 확인해요.
2. 일예산이 이미 소진됐는지 확인해요.
3. 연결된 이벤트가 승인 상태인지 확인해요.
4. 광고 지역과 이벤트 카테고리가 맞는지 확인해요.
5. 최근 수정으로 재검수가 필요한 상태인지 확인해요.

## 이벤트가 반려됐어요

| 반려 사유 | 해결 방법 |
| --- | --- |
| 가격 정보 불일치 | 이미지, 제목, 옵션, VAT 입력값을 같은 기준으로 맞춰요 |
| 부작용 안내 누락 | 부작용과 개인차 안내를 추가해요 |
| 보장 표현 사용 | 보장·확정 표현을 가능성·개인차 표현으로 바꿔요 |
| 이미지 기준 위반 | 선정적 이미지나 권한이 불명확한 이미지를 교체해요 |
| 카테고리 불일치 | 실제 수술·상담 내용과 맞는 카테고리로 수정해요 |

## 상담이 누락돼요

1. 상담 담당자와 응대 가능 시간을 다시 정해요.
2. 부재중 콜백 기준을 병원 내부에 공유해요.
3. 미응답 채팅과 콜백 요청을 먼저 확인해요.
4. 상담 후 예약 여부와 다음 안내를 기록해요.

## 예약 시간이 맞지 않아요

진료시간, 점심시간, 휴무일, 임시 휴진일을 먼저 확인해요. 병원 내부 예약표와 플랫폼 예약 가능 시간이 다르면 환자 불만이 생길 수 있어요.
