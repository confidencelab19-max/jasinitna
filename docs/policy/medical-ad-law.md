자신있나에 등록하는 병원 소개, 이벤트, 이미지, 광고 문구는 의료광고 기준을 지켜야 해요. 이 문서는 병원이 콘텐츠를 등록하기 전에 스스로 점검할 수 있도록 정리한 안내예요.

## 등록 전에 먼저 확인해요

| 확인 항목 | 점검 방법 |
| --- | --- |
| 사실 여부 | 병원명, 주소, 진료시간, 의료진 정보, 이벤트 기간, 가격, 옵션이 실제 운영 내용과 같은지 확인해요 |
| 가격 조건 | VAT 포함 여부, 추가 비용 가능성, 옵션별 금액, 적용 제외 조건을 함께 적어요 |
| 효과 표현 | 확대, 개선, 회복, 만족, 통증 같은 표현은 개인차가 있다는 전제로 작성해요 |
| 비교 표현 | 다른 병원보다 우수하다는 표현, 근거 없는 1위·최고·유일 표현은 쓰지 않아요 |
| 후기·사례 | 환자의 치료 경험담, 전후 비교, 특정 결과를 보장하는 사례처럼 보이지 않게 관리해요 |
| 이미지 | 신체 부위를 과도하게 강조하거나 불쾌감을 줄 수 있는 이미지는 쓰지 않아요 |
| 상담 안내 | 앱에 보이는 조건과 상담에서 말하는 가격·기간·옵션이 같아야 해요 |

## 등록 전에 문구를 이렇게 바꿔요

| 위험한 표현 | 바꾸는 방향 |
| --- | --- |
| 부작용 없는 수술 | 부작용 가능성이 있어 상담이 필요해요 |
| 통증 없이 회복 | 통증과 회복 기간은 개인차가 있어요 |
| 무조건 만족 | 결과는 개인 상태에 따라 달라질 수 있어요 |
| 최저가 보장 | 가격 조건과 포함 항목을 구체적으로 안내해요 |
| 국내 최고 | 확인 가능한 사실 중심으로 작성해요 |
| 빠른 회복만 강조하는 표현 | 치료 과정, 회복 기간, 개인차를 함께 안내해요 |
| 효과를 보장하는 표현 | 기대할 수 있는 방향과 개인별 차이를 구분해요 |
| 흉터 없음 | 흉터 가능성, 회복 과정, 상담 필요성을 함께 안내해요 |

## 이벤트 가격을 쓸 때 지켜요

가격은 환자가 가장 민감하게 보는 정보예요. 낮은 금액만 크게 보이고 실제 상담에서 조건이 달라지면 신뢰가 떨어져요.

| 항목 | 작성 기준 |
| --- | --- |
| 대표가 | 환자가 가장 먼저 보는 가격이에요. 적용 조건이 있으면 같이 적어요 |
| VAT | 포함 또는 별도 여부를 분명히 표시해요 |
| 옵션가 | 재료, 범위, 마취, 사후관리 등 옵션별 차이를 나눠 적어요 |
| 추가 비용 | 진료 후 개인 상태에 따라 달라질 수 있는 비용은 미리 안내해요 |
| 기간 | 이벤트 시작일, 종료일, 적용 요일이나 시간 조건을 정확히 적어요 |

## 이미지와 배너를 올릴 때 확인해요

- 이미지 안의 가격, 기간, 문구가 입력한 이벤트 내용과 같아야 해요.
- 성기사진, 과도한 신체 노출, 혐오감을 줄 수 있는 이미지는 쓰지 않아요.
- 의료진 사진과 병원 내부 사진은 실제 병원 자료만 사용해요.
- 전후 비교처럼 특정 결과를 기대하게 만드는 이미지는 사용 전 별도 검토가 필요해요.
- 이미지에 들어간 문구도 검수 대상이에요. 본문을 고쳐도 이미지 문구가 그대로면 반려될 수 있어요.

## 의료진·병원 소개를 쓸 때 확인해요

| 항목 | 작성 기준 |
| --- | --- |
| 의료진 경력 | 확인 가능한 경력과 자격만 적어요 |
| 진료 분야 | 실제 진료하는 범위 안에서 작성해요 |
| 장비·시설 | 보유 여부와 사용 범위를 과장하지 않아요 |
| 진료시간 | 실제 예약 가능한 시간과 같게 유지해요 |
| 위치·연락처 | 환자가 내원 전 확인할 수 있게 최신 정보로 관리해요 |

## 검수 반려를 줄이는 방법

1. 가격, VAT, 옵션, 추가 비용을 먼저 맞춰요.
2. 효과 보장, 통증 없음, 부작용 없음, 최고·유일 표현을 빼요.
3. 이미지 안 문구와 본문 문구가 같은지 확인해요.
4. 상담팀이 실제로 안내할 수 있는 조건인지 확인해요.
5. 반려 사유가 오면 같은 표현이 다른 위치에 남아 있는지 함께 봐요.

## 공식 근거

의료광고와 환자 유인·알선 관련 기준은 국가법령정보센터의 의료법을 기준으로 확인해요. 자세한 법령 해석이 필요한 경우 병원 내부 법무 또는 전문기관 확인이 필요해요.

- 의료법 제56조: 의료광고의 금지 기준
- 의료법 제57조: 의료광고 심의 관련 기준
- 의료법 제27조: 무면허 의료행위 등 금지와 환자 유인·알선 제한
