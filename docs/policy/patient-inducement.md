플랫폼 운영 중에는 환자를 부당하게 유인하거나 알선으로 오해될 수 있는 행동을 피해야 해요. 자신있나는 병원이 정보를 정확하게 등록하고, 환자가 상담과 예약을 판단할 수 있도록 돕는 가이드를 제공해요.

## 하면 안 돼요

- 현금, 상품권, 리워드 제공을 조건으로 상담이나 예약을 유도하면 안 돼요.
- 후기 작성을 조건으로 혜택을 제공하면 안 돼요.
- 특정 환자에게 반복적으로 광고성 연락을 보내면 안 돼요.
- 플랫폼에서 확인한 환자정보를 외부 영업이나 광고 문자 발송에 쓰면 안 돼요.
- 실제와 다른 가격이나 조건으로 상담을 유도하면 안 돼요.
- 환자에게 특정 치료를 받도록 압박하거나 예약을 강요하면 안 돼요.
- 상담 목적을 벗어나 보호자, 지인, 외부 업체에 환자정보를 전달하면 안 돼요.

## 안전하게 운영해요

| 상황 | 운영 기준 |
| --- | --- |
| 이벤트 문의 | 이벤트 가격, 포함 항목, 제외 조건을 그대로 안내해요 |
| 예약 문의 | 가능한 시간과 변경 기준을 명확히 안내해요 |
| 후기 요청 | 후기 작성은 환자의 자유로운 선택으로 남겨요 |
| 재상담 안내 | 상담 목적과 필요한 범위 안에서만 연락해요 |
| 광고 문자 | 별도 동의와 관련 법령 기준을 확인해요 |

## 상담에서 특히 조심해요

| 상황 | 안전한 안내 |
| --- | --- |
| 가격을 묻는 경우 | 앱에 등록된 가격, VAT, 옵션, 추가 비용 가능성을 같은 기준으로 설명해요 |
| 결과를 묻는 경우 | 개인 상태에 따라 결과와 회복이 달라질 수 있다고 안내해요 |
| 예약을 망설이는 경우 | 혜택을 조건으로 압박하지 말고 상담 가능 시간과 절차만 안내해요 |
| 후기 작성을 묻는 경우 | 후기는 환자 선택이며 대가를 조건으로 요청하지 않아요 |
| 사진을 보내는 경우 | 상담 목적에 필요한 범위에서만 확인하고 외부 저장이나 재사용은 하지 않아요 |

## 환자정보 사용 기준

파트너센터에서 확인한 연락처, 상담 내용, 예약 정보, 첨부 이미지는 상담과 예약 처리를 위해서만 사용해요.

- 담당자 개인 메신저로 옮기지 않아요.
- 광고 문자, 이벤트 홍보, 외부 영업에 쓰지 않아요.
- 상담이 끝난 뒤에도 불필요하게 반복 연락하지 않아요.
- 퇴사자와 외부 대행사가 환자정보를 볼 수 없게 계정을 정리해요.
- 민감한 이미지는 다운로드하거나 별도 보관하지 않아요.

## 내부 기준을 이렇게 맞춰요

1. 상담 담당자는 앱에 노출된 이벤트 조건을 기준으로 답변해요.
2. 예약 담당자는 실제 가능한 시간만 확정해요.
3. 마케팅 담당자는 혜택 문구를 상담·예약 조건처럼 쓰지 않아요.
4. 후기 담당자는 환자 개인정보와 진료 내용을 공개 답글에 쓰지 않아요.
5. 정산 담당자는 광고비와 이벤트 비용을 환자 비용처럼 혼동해 안내하지 않아요.

## 문제가 생기기 쉬운 경우

상담 수를 늘리기 위해 혜택을 약속하거나, 환자정보를 다른 목적으로 사용하는 경우 문제가 될 수 있어요. 병원 내부 담당자는 상담 목적, 연락 근거, 안내 문구를 항상 같은 기준으로 관리해 주세요.
