이미지와 후기, 상담 자료는 환자가 빠르게 신뢰를 판단하는 요소예요. 동시에 의료광고와 개인정보 이슈가 생기기 쉬운 영역이어서 등록 전 기준을 분명히 해야 해요.

## 병원 이미지

| 사용할 수 있는 방향 | 피해야 할 방향 |
| --- | --- |
| 실제 병원 외관, 접수 공간, 상담실, 진료실을 보여줘요 | 실제와 다른 장소, 과도하게 보정된 이미지, 출처가 불명확한 이미지는 쓰지 않아요 |
| 환자가 내원 전 참고할 수 있는 정보를 보여줘요 | 시설 수준을 과장하거나 보유하지 않은 장비처럼 보이게 하지 않아요 |

## 의료진 이미지

의료진 사진은 환자에게 신뢰를 주지만, 경력과 자격을 과장하면 문제가 될 수 있어요.

- 실제 의료진 사진만 사용해요.
- 경력, 학회, 자격, 수상 내용은 확인 가능한 정보만 적어요.
- 전문성을 표현하더라도 결과를 보장하는 문장과 연결하지 않아요.
- 의료진이 아닌 사람이 의료진처럼 보이는 이미지는 쓰지 않아요.

## 신체 이미지와 수술 관련 이미지

남성수술 플랫폼 특성상 신체 관련 이미지의 기준을 더 보수적으로 봐야 해요.

- 성기사진, 과도한 노출, 불쾌감을 줄 수 있는 이미지는 쓰지 않아요.
- 수술 결과를 보장하는 전후 비교처럼 보이는 구성은 사용하지 않아요.
- 환자 사진, 상담 사진, 진료 자료는 광고 이미지로 임의 사용하지 않아요.
- 설명을 위해 필요한 경우에도 환자 식별 가능성이 있으면 제외해요.

## 후기와 상담 내용을 다룰 때

후기 답글이나 상담 내용은 공개 문구가 되면 의료광고처럼 보일 수 있어요. 환자 개인정보, 진료 내용, 수술 결과를 공개 답글에 쓰지 않아요.

| 상황 | 안전한 처리 |
| --- | --- |
| 좋은 후기에 답글을 달아요 | 감사 인사와 일반적인 안내만 남겨요 |
| 불만 후기에 답글을 달아요 | 개인정보와 진료 내용을 공개하지 말고 확인 가능한 연락 경로를 안내해요 |
| 후기 작성을 요청하고 싶어요 | 대가나 혜택을 조건으로 요청하지 않아요 |
| 상담 이미지를 받았어요 | 상담 목적에 필요한 범위에서만 확인하고 외부 저장하지 않아요 |

## 환자자료를 보관하지 않아요

파트너센터에서 본 환자 연락처, 상담 내용, 첨부 이미지는 상담·예약 처리 목적 안에서만 사용해요. 개인 메신저, 개인 저장소, 외부 업체, 광고 소재 제작 목적으로 옮기지 않아요.

## 담당자에게 공유할 기준

1. 이미지는 실제 병원과 실제 의료진 자료만 사용해요.
2. 환자 신체나 상담 자료는 광고 소재로 쓰지 않아요.
3. 후기 답글에는 개인정보와 진료 내용을 쓰지 않아요.
4. 가격과 조건이 이미지 안에 들어가면 입력값과 반드시 맞춰요.
5. 애매하면 등록하지 말고 의료광고 체크리스트부터 다시 확인해요.
