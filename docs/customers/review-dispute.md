후기가 마음에 들지 않는다는 이유만으로 삭제할 수는 없어요. 다만 개인정보 노출, 욕설, 명백한 허위사실, 타인 비방처럼 검토가 필요한 경우에는 자신있나에 확인을 요청할 수 있어요.

## 검토 요청 전 확인해요

| 상황 | 요청 가능성 |
| --- | --- |
| 환자 연락처나 실명이 노출돼요 | 검토 요청이 필요해요 |
| 성적 모욕, 욕설, 혐오 표현이 있어요 | 검토 요청이 필요해요 |
| 병원 방문 사실과 다른 내용이라고 판단돼요 | 근거를 정리해 요청해요 |
| 단순 불만이나 낮은 평점이에요 | 답글 또는 재상담 안내를 먼저 검토해요 |
| 의료 결과에 대한 주관적 평가예요 | 삭제보다 사실관계 안내가 적절할 수 있어요 |

## 요청할 때 정리할 내용

- 후기 작성일과 작성 위치
- 문제가 되는 문장 또는 개인정보 항목
- 병원에서 확인한 사실관계
- 공개 답글로 해결 가능한지 여부
- 환자에게 재상담을 안내했는지 여부

## 답글 작성 기준

답글에는 환자의 진료 정보, 상담 내용, 내원 여부를 직접 드러내지 않아요. 공개적으로 다투기보다 확인 가능한 범위에서 안내하고, 필요한 경우 비공개 상담 경로로 연결해요.
