상담팀이 앱에 보이는 정보와 다른 내용을 말하면 환자는 가격이나 조건을 잘못 이해할 수 있어요. 상담 문구는 이벤트 등록 내용과 같은 기준으로 맞춰야 해요.

## 상담 전 확인해요

1. 환자가 본 이벤트 제목을 확인해요.
2. 대표 가격과 옵션 가격을 확인해요.
3. VAT 포함 여부를 확인해요.
4. 추가 비용이 발생할 수 있는 조건을 확인해요.
5. 수술 가능 여부는 상담 후 달라질 수 있음을 안내해요.

## 안내 문구 방향

| 위험한 안내 | 안전한 안내 |
| --- | --- |
| 누구나 이 가격으로 가능해요 | 개인 상태와 선택 옵션에 따라 달라질 수 있어요 |
| 부작용은 없어요 | 부작용 가능성이 있어 상담이 필요해요 |
| 바로 회복돼요 | 회복 기간은 개인차가 있을 수 있어요 |
| 무조건 만족해요 | 결과는 개인 상태에 따라 달라질 수 있어요 |

## 병원 내부 공유 방법

이벤트가 수정되면 상담팀에 변경된 가격, 옵션, 적용 기간을 바로 공유해요. 광고가 집행 중인 이벤트는 특히 상담 문구와 화면 문구가 달라지지 않게 확인해요.
