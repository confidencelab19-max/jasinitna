비뇨기과·남성수술 상담은 환자가 통화 가능한 시간이 짧을 수 있어요. 부재중 상담을 놓치면 같은 광고비로 얻은 상담 기회가 사라질 수 있어요.

## 콜백 기준을 정해요

| 상황 | 권장 처리 |
| --- | --- |
| 첫 통화 부재 | 일정 시간 뒤 1회 재연락해요 |
| 두 번째 부재 | 문자 또는 채팅으로 상담 가능 시간을 안내해요 |
| 환자가 시간 요청 | 요청 시간 전후로 담당자를 지정해요 |
| 진료시간 외 문의 | 다음 운영 시작 시간에 먼저 확인해요 |
| 예약 직전 문의 | 예약 가능 여부를 먼저 확인해요 |

## 담당자에게 보여야 하는 정보

- 환자가 남긴 연락 가능 시간
- 환자가 본 이벤트와 가격
- 이전 통화 시도 시간
- 다음 콜백 예정 시간
- 예약 연결 여부

## 하지 않아야 해요

환자 동의 없이 반복적으로 광고성 문자를 보내면 안 돼요. 상담 목적을 넘어선 재연락은 환자 불편과 법적 리스크로 이어질 수 있어요.
