자신있나 운영은 하루에 오래 붙잡고 보는 것보다, 정해진 시간에 꼭 필요한 항목을 빠르게 확인하는 방식이 좋아요.

## 오전에 확인해요

- 밤사이 들어온 새 상담 요청
- 새 예약 요청과 예약 변경 요청
- 광고잔고 부족 알림
- 이벤트 반려 또는 수정 요청
- 오늘 예약 가능 시간과 병원 내부 예약표

## 오후에 확인해요

- 미응답 채팅
- 콜백 필요 상담
- 예약 확정 대기 상태
- 광고 리포트의 상담 연결 흐름
- 환자 문의가 반복되는 이벤트 문구

## 마감 전에 확인해요

- 오늘 예약 상태 정리
- 방문완료, 노쇼, 취소 상태 반영
- 후기 답글 필요 여부
- 내일 휴진·마감 시간 반영
- 광고잔고와 일예산 상태

## 담당자별로 나눠요

상담 담당자는 미응답과 콜백을 보고, 데스크는 예약 상태를 보고, 마케팅 담당자는 이벤트와 광고를 보고, 정산 담당자는 충전과 증빙을 확인해요. 한 사람이 모두 보더라도 화면별 확인 순서를 정해두면 누락을 줄일 수 있어요.
