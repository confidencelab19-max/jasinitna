자신있나는 성인 남성 전용 비뇨기과·남성수술 플랫폼이에요. 병원은 파트너센터에서 병원 정보, 이벤트, 상담, 예약, 광고, 후기 답글을 운영할 수 있어요.

처음 입점했다면 아래 순서대로 설정해 주세요.

## 1. 병원 기본정보를 확인해 주세요

병원명, 주소, 대표 전화번호, 진료시간, 휴무일, 상담 가능 시간이 실제 운영 정보와 같은지 확인해 주세요. 환자는 이 정보를 보고 상담과 예약을 결정해요.

## 2. 상담 담당자와 콜백 기준을 정해 주세요

전화상담과 채팅상담은 응답 속도가 중요해요. 담당자, 응대 가능 시간, 부재중 재연락 기준을 병원 내부에서 먼저 정해 주세요.

## 3. 이벤트를 등록해 주세요

이벤트는 환자가 가장 많이 비교하는 콘텐츠예요. 가격, VAT, 옵션, 추가 비용, 부작용, 적용 조건을 정확히 입력해야 검수가 빠르게 진행돼요.

## 4. 광고를 시작하기 전 잔고와 랜딩을 확인해 주세요

광고는 승인된 이벤트 또는 병원 정보로 연결돼야 해요. 광고잔고, 무료·유료 포인트, 일예산, 광고 지역을 먼저 확인해 주세요.

## 5. 후기와 환자정보 처리 기준을 공유해 주세요

후기 답글은 공개 응대예요. 환자 전화번호, 상담 내용, 신체사진, 성기사진 등 민감자료는 상담·예약 목적 외로 사용할 수 없어요.

## 매일 확인하면 좋아요

| 시간 | 확인할 것 |
| --- | --- |
| 오전 | 새 상담 요청, 새 예약, 광고잔고 부족 알림 |
| 오후 | 미응답 채팅, 콜백 누락, 이벤트 반려 사유 |
| 마감 전 | 예약 상태, 후기 답글, 광고 리포트 |
