환불과 보상은 광고 상품, 집행 상태, 취소 시점, 광고 소재 상태, 노출 여부를 확인한 뒤 처리돼요. 파트너센터에서는 요청 전에 필요한 정보를 먼저 정리하면 확인이 빨라져요.

## 요청 전에 확인해요

| 확인 항목 | 병원이 준비할 내용 |
| --- | --- |
| 광고 상품 | 어떤 광고 신청 건인지, 계약 안내와 파트너센터 상태가 일치하는지 확인해요 |
| 집행 상태 | 승인 대기, 집행 예정, 집행 중, 종료 중 어느 상태인지 확인해요 |
| 취소 사유 | 단순 취소, 일정 변경, 소재 수정, 검수 반려 등 사유를 정리해요 |
| 사용 내역 | 이미 사용된 광고비와 남은 잔고를 확인해요 |
| 증빙 | 결제 내역, 세금계산서, 담당자 연락처를 준비해요 |

## 처리 흐름

1. 파트너센터에서 광고 상품과 집행 상태를 확인해요.
2. 취소 또는 환불 요청 사유를 정리해요.
3. 자신있나 운영팀이 사용 내역과 광고 상태를 확인해요.
4. 환불 가능 여부와 처리 방식을 안내받아요.
5. 처리 결과를 병원 내부 정산 담당자에게 공유해요.

## 무료 포인트 환불

무료 지급 포인트는 현금 환불 대상이 아니에요. 프로모션, 입점 혜택, 보상 목적으로 지급된 포인트는 유효기간과 사용 가능 상품을 확인해 주세요.
