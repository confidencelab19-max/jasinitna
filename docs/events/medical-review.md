이벤트는 의료광고 기준을 지켜야 노출될 수 있어요. 자신있나는 환자 오인 가능성이 있는 문구와 이미지를 검수해요.

## 공통 검수 기준

| 기준 | 확인 내용 |
| --- | --- |
| 사실성 | 실제 병원 정보와 일치하는지 확인해요 |
| 명확성 | 가격, VAT, 조건, 기간, 수량을 숨기지 않았는지 확인해요 |
| 비보장성 | 효과, 통증, 부작용, 회복 기간을 단정하지 않았는지 확인해요 |
| 비선정성 | 신체 부위를 과도하게 강조하지 않았는지 확인해요 |
| 개인정보 | 환자 사진, 후기, 상담 이미지를 동의 없이 쓰지 않았는지 확인해요 |

## 사용하지 않는 것이 좋아요

- 부작용 없음
- 통증 없음
- 흉터 없음
- 100% 만족
- 무조건 효과
- 국내 1위
- 최저가 보장
- 당일 완벽 회복
- 실패 없는 수술

## 더 안전한 표현

| 기존 표현 | 수정 방향 |
| --- | --- |
| 통증 없는 수술 | 통증은 개인차가 있을 수 있어요 |
| 부작용 없음 | 부작용 가능성이 있어 상담이 필요해요 |
| 만족 보장 | 결과는 개인 상태에 따라 달라질 수 있어요 |
| 당일 회복 | 회복 기간은 개인차가 있을 수 있어요 |

## 변경 시 다시 검수될 수 있어요

승인된 이벤트라도 제목, 가격, 이미지, 옵션, 랜딩 내용이 바뀌면 다시 검수될 수 있어요. 광고에 연결된 이벤트라면 재승인 전까지 광고 노출이 제한될 수 있어요.
