이벤트는 운영 중에도 가격, 이미지, 기간, 옵션을 바꿀 수 있어요. 다만 환자에게 보이는 정보가 바뀌는 수정은 다시 확인이 필요할 수 있어요.

## 수정 전에 확인해요

| 수정 항목 | 확인할 내용 |
| --- | --- |
| 가격 | 대표 가격, 옵션 가격, VAT 표시가 모두 같은지 확인해요 |
| 기간 | 종료일, 상시 여부, 요일 조건을 확인해요 |
| 이미지 | 이미지 속 가격과 문구가 입력값과 같은지 확인해요 |
| 상세 설명 | 부작용, 개인차, 추가 비용 가능성을 빠뜨리지 않아요 |
| 카테고리 | 실제 수술·상담 내용과 맞는지 확인해요 |

## 중지해야 하는 경우

- 이벤트 기간이 끝났어요.
- 가격이나 옵션이 병원 내부 운영과 달라졌어요.
- 상담팀이 더 이상 해당 조건으로 안내할 수 없어요.
- 의료광고 기준상 수정이 필요한 문구가 발견됐어요.
- 연결된 광고를 잠시 멈춰야 해요.

## 재노출 전 체크리스트

1. 최신 가격과 상담 안내가 일치하는지 확인해요.
2. 이미지 속 문구가 오래된 조건을 담고 있지 않은지 확인해요.
3. 반려 이력이 있다면 같은 표현이 다시 들어가지 않았는지 확인해요.
4. 광고 랜딩으로 쓰는 이벤트라면 광고 신청 화면에서도 연결 상태를 확인해요.
