이벤트 성과는 가격만으로 결정되지 않아요. 환자가 앱에서 본 내용과 상담에서 듣는 내용이 일치해야 전환율이 안정적으로 올라가요.

## 조회는 많은데 상담이 적을 때

- 대표 이미지가 이벤트 내용을 잘 보여주는지 확인해요.
- 가격과 옵션이 너무 복잡하지 않은지 확인해요.
- 추가 비용 안내가 과도하게 불안하게 보이지 않는지 확인해요.
- 제목과 이미지 문구가 서로 맞는지 확인해요.

## 상담은 많은데 예약이 적을 때

- 콜백 속도를 확인해요.
- 상담 담당자가 이벤트 조건을 정확히 알고 있는지 확인해요.
- 앱에 표시된 가격과 상담 가격이 일치하는지 확인해요.
- 예약 가능 시간이 충분히 열려 있는지 확인해요.

## 반려가 반복될 때

반려 사유를 단순히 문구만 바꿔 해결하지 말고, 가격·이미지·옵션·부작용 안내가 전체적으로 일치하는지 함께 확인해 주세요.
