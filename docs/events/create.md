이벤트는 자신있나에서 환자가 가장 많이 비교하는 콘텐츠예요. 등록 후 바로 노출되는 것이 아니라, 자신있나 검수 후 승인된 이벤트만 노출돼요.

## 등록 순서

1. 이벤트 카테고리를 선택해요.
2. 이벤트 제목을 입력해요.
3. 대표 가격과 옵션 가격을 입력해요.
4. VAT 포함 여부를 표시해요.
5. 적용 기간과 대상 조건을 입력해요.
6. 이벤트 이미지와 상세 이미지를 등록해요.
7. 수술·시술 설명과 부작용 안내를 입력해요.
8. 검수 요청을 보내요.

## 등록 전 확인할 내용

| 항목 | 확인 기준 |
| --- | --- |
| 이벤트명 | 실제 시술·수술명과 맞아야 해요 |
| 카테고리 | 포경, 정관, 확대, 조루, 발기부전 등 내용과 맞게 선택해요 |
| 가격 | VAT, 옵션, 추가 비용 가능성을 함께 적어요 |
| 기간 | 상시, 기간 한정, 요일 한정 조건을 명확히 적어요 |
| 이미지 | 제목, 가격, 옵션 정보와 서로 맞아야 해요 |
| 부작용 | 대표 부작용과 개인차 안내를 포함해요 |

## 자주 반려되는 경우

- 이벤트 제목과 이미지 문구가 달라요.
- 가격에는 추가 비용이 있는데 이미지에는 추가 비용이 없는 것처럼 보여요.
- VAT 포함 여부가 빠져 있어요.
- 이벤트 카테고리가 실제 수술 내용과 달라요.
- 부작용 안내가 없거나 너무 작게 표시돼요.
