남성수술 이벤트는 개인 상태, 재료, 옵션, 마취, 사후관리 여부에 따라 최종 비용이 달라질 수 있어요. 가격 정보는 환자가 오해하지 않도록 충분히 구체적으로 적어 주세요.

## 가격에 꼭 들어가야 하는 내용

- VAT 포함 또는 별도 여부
- 기본가에 포함되는 항목
- 추가 비용이 발생할 수 있는 항목
- 옵션별 가격
- 적용 기간
- 적용 대상 조건
- 상담 후 최종 비용이 달라질 수 있는 사유

## 가격 표기 예시

| 상황 | 권장 표기 |
| --- | --- |
| VAT 포함 | 이벤트가 000원 VAT 포함 |
| VAT 별도 | 이벤트가 000원 VAT 별도 |
| 추가 비용 가능 | 개인 상태와 선택 옵션에 따라 추가 비용이 발생할 수 있어요 |
| 상담 후 결정 | 상담 후 적합한 수술 방법과 최종 비용이 결정돼요 |

## 확인해 주세요

이미지에 적은 가격, 옵션명, 횟수, 이벤트 기간이 파트너센터에 입력한 정보와 같아야 해요. 서로 다르면 검수 반려 또는 환자 민원이 발생할 수 있어요.
