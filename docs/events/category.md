카테고리는 이벤트 노출 위치와 광고 과금 기준에 영향을 줄 수 있어요. 실제 수술 내용과 가장 가까운 카테고리를 선택해 주세요.

## 예시 카테고리

| 구분 | 예시 |
| --- | --- |
| 기본 상담성 | 포경수술, 정관수술, 단순 상담 |
| 일반 남성수술 | 조루, 귀두, 여유증, 모발이식, 기타 |
| 고가 남성수술 | 음경확대, 길이연장, 대체진피, 자가지방, 진피이식, 발기부전 수술 |

## 주의할 점

고가 남성수술 이벤트를 낮은 카테고리로 등록하면 검수에서 수정 요청을 받을 수 있어요. 카테고리가 맞지 않으면 환자에게 잘못 노출되거나 광고 효율이 왜곡될 수 있어요.

## 카테고리를 바꿔야 하는 경우

- 이벤트 제목과 카테고리가 맞지 않아요.
- 옵션이 실제로는 다른 수술에 가까워요.
- 광고 성과가 특정 카테고리에서 비정상적으로 낮아요.
- 자신있나 검수에서 카테고리 수정 요청을 받았어요.
