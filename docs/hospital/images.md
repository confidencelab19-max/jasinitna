병원 이미지는 환자가 병원 분위기와 신뢰도를 판단하는 자료예요. 선정적이거나 의료광고 기준에 맞지 않는 이미지는 사용할 수 없어요.

## 권장 이미지

- 병원 외관
- 접수 공간
- 상담실
- 진료실
- 의료진 단체 이미지
- 장비 이미지
- 내원 안내 이미지

## 사용을 피해야 하는 이미지

- 성기나 음모가 직접 노출되는 이미지
- 신체 부위를 과도하게 확대해 보여주는 이미지
- 수술 전후를 보장하는 것처럼 보이는 비교 이미지
- 환자가 상담 중 보낸 신체사진
- 사용 권한이 불명확한 이미지
- 과도하게 선정적인 모델 이미지

## 이미지에 문구를 넣을 때

이미지 안의 문구도 의료광고 검수 대상이에요. 가격, 이벤트명, VAT, 옵션, 기간이 들어간 경우 등록 정보와 반드시 일치해야 해요.
